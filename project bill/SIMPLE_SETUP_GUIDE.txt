HOW TO CREATE A BILL (Step-by-Step)
================================================================================

When the software opens, you will see a green form:

1. BILL NUMBER
   ‚Üí Already filled automatically (don't change it)

2. DATE
   ‚Üí Today's date is already filled (you can change if needed)

3. CUSTOMER NAME
   ‚Üí Type customer's full name

4. CELL/TELE NO
   ‚Üí Type phone number

5. VEHICLE MAKE
   ‚Üí Type car brand and model (Example: Maruti Swift)

6. VEHICLE NO
   ‚Üí Type vehicle registration number (Example: KA-09-MN-1234)

7. ADD ITEMS/SERVICES:

   Row 1 is already there:
   - Particulars: Type what work was done (Example: Engine Oil Change)
   - Qty: Type quantity (Example: 1)
   - Amount: Type price per item (Example: 500)

   To add more items:
   - Click "+ Add Item" button at the bottom
   - Fill the new row

   To remove an item:
   - Click "Remove" button on that row

8. AUTOMATIC CALCULATIONS:

   As you type amounts, these will calculate automatically:
   - Basic Amount (sum of all items)
   - SGST 9% (tax)
   - CGST 9% (tax)
   - Total (Basic + SGST + CGST = 18% GST included)
   - Amount in Words (in English)

9. GENERATE PDF:

   When everything is filled:
   - Click the big green "GENERATE PDF" button
   - You'll see a message "PDF saved successfully"
   - Bill number will increase automatically for next bill
   - Form will clear automatically

10. VIEW YOUR BILLS:

    - Click "Open Bills Folder" button at the top
    - All your bills are saved here as PDF files
    - You can print these PDFs anytime

COMMON PROBLEMS & SOLUTIONS
================================================================================

PROBLEM: "npm is not recognized" error
SOLUTION:
  - Node.js is not installed properly
  - Go back to STEP 1 and install Node.js again
  - Make sure to RESTART computer after installing


PROBLEM: Application doesn't start
SOLUTION:
  - Make sure you ran "npm install" first (STEP 2)
  - Try closing and opening again
  - Restart your computer and try again


PROBLEM: Can't find START_APP.bat file
SOLUTION:
  - Make sure you're in the correct folder
  - Folder should be: "Downloads" ‚Üí "fixplus" ‚Üí "project bill"
  - If still not there, use METHOD 2 to start


PROBLEM: PDF not generating
SOLUTION:
  - Make sure all fields are filled (marked with red)
  - Make sure amounts are numbers only (no letters)
  - Try clicking Generate PDF button again


PROBLEM: Bill numbers not increasing
SOLUTION:
  - This is normal - bill number increases AFTER you generate a PDF
  - If stuck, close and reopen the application
